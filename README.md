# 臭蛋

### 实现功能

  1. 首页显示最近四篇文章
  2. 实现 分类，标签，归档
  3. 在 单独文章 页面点击 分类 或 标签 时，连接到所属 分类 或 标签的页面。
  4. 文章页面自动识别 h3 标签，并在窗口底部现实导航条
  5. CSS 完全兼容 IE7 以上浏览器，自然也兼容其它浏览器了。IE6 整体布局较完整
  6. JavaScript 兼容所有浏览器。代码高亮使用 highlight.js，官方支持 IE8 以上浏览器
  7. 添加 多说评论 和 百度分享
